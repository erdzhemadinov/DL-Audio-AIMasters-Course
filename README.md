# Deep Learning for Audio Course, 2023

## Description
Topics discussed in course:
- Digital Signal Processing
- Automatic Speech Recognition (ASR)
- Key-word spotting (KWS)
- Text-to-Speech (TTS)
- Voice Conversion
- Music Generation with NNs
- Unsupervised learning in Audio
